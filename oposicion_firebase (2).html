Configurar Firebase
Sigue estos 8 pasos, tarda unos 10 minutos y es completamente gratis.
1
Crear proyecto Firebase
Ve a console.firebase.google.com → Añadir proyecto → ponle un nombre (ej: "oposicion-2028") → desactiva Analytics → Crear.
2
Registrar la app web
En tu proyecto → icono </> (Web) → ponle un nombre → Registrar app. Copia el objeto firebaseConfig que aparece.
3
Pegar la configuración en este archivo
Abre este archivo .html con un editor de texto (Notepad, etc). Busca la sección firebaseConfig y reemplaza los valores de ejemplo por los tuyos.
4
Activar inicio de sesión con Google
En Firebase Console → Authentication → Comenzar → pestaña Proveedores → Google → activar → guardar.
5
Crear la base de datos Firestore
En Firebase Console → Firestore Database → Crear base de datos → Modo producción → elige región europe-west1 (o la más cercana) → Crear.
6
Configurar las reglas de seguridad
En Firestore → pestaña Reglas → borra el contenido y pega esto → Publicar:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /users/{userId}/{document=**} { allow read, write: if request.auth != null && request.auth.uid == userId; } } }
7
Añadir tu dominio autorizado
Si abres el archivo directamente desde tu ordenador, ve a Firebase Console → Authentication → Configuración → Dominios autorizados → añade localhost. Si usas GitHub Pages u otro hosting, añade ese dominio.
8
¡Listo! Abre el archivo y accede con Google
Abre el archivo .html en Chrome, pulsa Continuar con Google y tu progreso se guardará automáticamente en la nube cada vez que hagas un cambio.
